## Getting Started

### Local Development

```bash
npm install
npm run dev
```

Visit [http://localhost:3000](http://localhost:3000)

### Build for Production

```bash
npm run build
npm start
```

## Deployment to Vercel (FREE)

The easiest way to deploy this website is using [Vercel](https://vercel.com):

1. Push this folder to GitHub
2. Go to [vercel.com](https://vercel.com)
3. Click "New Project"
4. Import your GitHub repository
5. Vercel will auto-detect Next.js and deploy automatically
6. Your site is now live at your-domain.vercel.app

### Connect Your Custom Domain

1. In Vercel dashboard, go to Settings
2. Add your custom domain (cruzer.app)
3. Follow DNS instructions
4. Domain will be active within 24 hours

- `app/page.tsx` - Main homepage content
- `app/globals.css` - All styling
- `app/layout.tsx` - HTML head and metadata
- `next.config.js` - Next.js configuration
